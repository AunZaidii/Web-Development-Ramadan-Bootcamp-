# Aun's Kitchen - Web Development Project

This project is a responsive food delivery and menu exploration website titled **Aun's Kitchen**. It features a modern layout with a hero section, service widgets, a categorized menu, and a newsletter footer.

## Features

* **Responsive Design**: Fully optimized for desktops, tablets (769px – 1024px), and mobile devices (up to 480px) using custom media queries.
* **Hero Section**: Highlights promotional offers such as "Upto 50% off on first order!" and branding.
* **Service Widgets**: Showcases key offerings including discount vouchers, fresh/healthy food, and fast home delivery.
* **Menu Exploration**: A grid layout showcasing diverse cuisines including Asian, Italian, Chinese, Mexican, French, and Korean.
* **Interactive UI**: Includes hover effects on navigation buttons and menu items for an enhanced user experience.
* **Newsletter & Socials**: Features a footer with a newsletter signup and links to Facebook, Instagram, and Twitter.

## Technologies Used

* **HTML5**: For semantic structuring of the web content.
* **CSS3**: For layout styling, animations, and responsiveness.
* **Font Awesome**: Integrated for scalable vector icons.
* **Google Fonts**: Utilizes "Nunito", "Bubblegum Sans", "Stoke", and "Fuzzy Bubbles" for typography.

## File Structure

* `index.html`: The main entry point containing the website structure.
* `styles.css`: Contains global styles, typography, and component styling.
* `responsiveness.css`: Dedicated stylesheet for handling media queries across different screen sizes.
* `/Logos/`: Directory containing branding assets like `logo.png` and `favicon.png`.
* `/Pictures/` & `/cuisines/`: Images used for the hero section and menu categories.
